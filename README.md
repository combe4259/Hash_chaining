# Hash Chaining - Manchester United Starting Lineup

이 프로젝트는 **해시 테이블(Hash Table)** 구조와 **체이닝(Chaining)** 기법을 **구현하면서 개념을 학습하기 위해 제작한 C++ 콘솔 프로그램**입니다.  
자료구조 수업에서 배우는 해시 충돌 처리 방식을 단순히 이론으로 익히는 것이 아니라, **직접 코드로 구현하고 실습하기 위해 시작했습니다.**

## 목적

- 해시 테이블의 기본 개념과 작동 원리를 이해
- 해시 충돌 시 체이닝 방식으로 연결 리스트를 활용하여 문제를 해결하는 구조 구현
- 해시 함수 설계 및 충돌 처리의 중요성을 체감하고 자료구조에 대한 학습

## 주요 기능

- 선수 이름과 등번호 데이터를 파일로부터 읽어들임
- 이름을 키로 등번호를 값으로 하여 해시 테이블에 저장
- 해시 충돌 발생 시 연결 리스트를 이용한 체이닝 기법으로 처리
- 이름이 중복될 경우 기존보다 더 큰 등번호로 갱신
- 사용자 입력을 통해 특정 선수의 등번호를 조회할 수 있음

## 학습한 점

- 문자열 기반 해시 함수 구현 방식
- 체이닝 구조(Linked List)로 충돌을 처리하는 로직
- 구조체와 포인터를 활용한 C++ 메모리 관리
- 텍스트 파일을 활용한 입력 처리 로직
